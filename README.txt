# Universal Data Cleaner & EDA Tool

A Streamlit-based application for automated data cleaning, profiling, exploratory data analysis (EDA), and SQL-style querying.

## Features
- Upload CSV / Excel / JSON files
- Automatic data profiling
- Rule-based data cleaning engine
- Missing value handling & outlier removal
- Interactive EDA visualizations
- SQL-style query editor
- Audit logs for transparency
- Export cleaned datasets
